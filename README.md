# kaqvuUtil

## Lokalne uruchomienie

```bash
npm install
npm run dev
```

Frontend będzie dostępny na http://localhost:5173

## Budowanie

```bash
npm run build
```

## Struktura projektu

```
/api          - Vercel API Routes (backend)
/src          - Frontend (React + Vite)
index.html    - Główny plik HTML
vercel.json   - Konfiguracja Vercel
```
